# grafika-V-sem

keybinds:

t - textury on/off

r - obrot piramidy on/off

e - obrot slonca on/off

j - zmiana koloru swiatla punktowego

n - swiatlo kierunkowe on/off

h - spojrzenie na srodek

SHIFT - kamera w dol

SPACE - kamera w gore

a - kamera w lewo

d - kamera w prawo

w - kamera do przodu

s - kamera do tylu

mysz - rozgladanie sie
